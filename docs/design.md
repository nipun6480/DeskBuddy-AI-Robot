# DeskBuddy AI Robot Design

## Goal

Create a small desk robot that interacts with the user through touch and messages.

## Core Modules

1. Interaction Module

   * Touch sensor input
   * User interaction

2. Display Module

   * OLED display
   * Time and messages

3. AI Interaction (Future)

   * Chat responses
   * Smart assistant features

## Hardware Architecture

ESP32 → Touch Sensor → Display → Speaker/Buzzer
